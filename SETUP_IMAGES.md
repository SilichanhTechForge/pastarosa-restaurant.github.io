## Need Help?

If the image doesn't show:
1. Make sure the filename is exactly `1.jpeg` (case-sensitive on some systems)
2. Verify it's in the `assets/images/` folder
3. Refresh the browser (Ctrl+R or Cmd+R)
4. Check the browser's Developer Console (F12) for errors
